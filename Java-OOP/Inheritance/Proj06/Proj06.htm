PROGRAM SPECIFICATIONS

Write a program named Proj06 that uses the driver class definition provided in the file named Proj06.java and Ericson's media library along with the image file named Proj06.jpg to produce the two graphic output images shown below.

Note that the images will both appear in the same location in the upper-left corner of the screen. You must move the top one in order to see the bottom one. Be sure to display your images in the order shown below. The second required output image covers the first required output image when the two images are first displayed on the screen.

Do not allow the size of the image to change when it is rotated.

You may define new classes as necessary to cause your program to behave as required, but you may not modify the class definition provided in the file named Proj06.java.

In addition to the output images mentioned above, your program must display your name on the command-line screen.
